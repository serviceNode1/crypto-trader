Safe Position Size =
(Account Size √ó Risk %) √∑ (Entry Price - Stop Loss Price)

Example:
- Account: $10,000
- Risk per trade: 2% = $200
- Entry: $50,000
- Stop Loss: $48,000
- Position Size: $200 √∑ $2,000 = 0.1 BTC
